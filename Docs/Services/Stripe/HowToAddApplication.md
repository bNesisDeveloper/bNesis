¹ Stripe registration

Added 11.07.2017.

Steps.

Step 1:  
Go to (https://dashboard.stripe.com/account/apikeys).

![image001](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Stripe/image001.png)

Step 2:  
Click “Reveal test key token” and get Api Secret test key.  
(https://dashboard.stripe.com/account/apikeys)

Step 3:  
Fill out all requested fields with verified information at Dashboard -> Activate Your Account.
(https://dashboard.stripe.com/account/details)

![image002](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Stripe/image002.png)
![image003](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Stripe/image003.png)

Step 4:  
Go to (https://dashboard.stripe.com/account/apikeys)  
and after that switch to Live Api keys with Viewing Test Data.
![image004](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Stripe/image004.png)

Step 5:  
Click “Reveal live key token” and get Secret Api live key.  
 (https://dashboard.stripe.com/account/apikeys)

![image005](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Stripe/image005.png)
