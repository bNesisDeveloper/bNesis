# Expensify registration  
Added 02.07.2018.

To use the API of Expensify, you will need to generate API credentials.

1. Create an Expensify account at (https://www.expensify.com/).

![image002](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image002.jpg)

2. Enter your e-mail and click Sign Up.

![image004](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image004.jpg)

3. You need to confirm your e-mail address in e-mail letters, which arrived in your e-mail inbox.

![image006](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image006.jpg)

4. After confirmation your e-mail address you need to set up your password for Expensify account.

![image008](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image008.jpg)

5. After set up your password you redirect in your Expensify account.

![image010](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image010.jpg)

6. For get a credentials (partnerUserID and partnerUserSecret) go to (https://www.expensify.com/tools/integrations/) and click “Click here”.

![image012](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image012.jpg)

7. A pair of credentials (partnerUserID and partnerUserSecret) will be generated and shown on the page. Make sure to store the partnerUserID and partnerUserSecret pair you’re given in a secure location, as you won’t be shown them again.

![image014](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Expensify/image014.jpg)
