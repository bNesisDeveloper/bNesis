AmoCRM registration  
Made on 08.10.2018.

To use the API of AmoCRM, you need to generate API credentials.

1. Go to the AmoCRM page (https://www.amocrm.ru/), enter you data and click 'Try it for free'.

![image002](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/AmoCRM/image002.jpg)

2. Enter information about yourself and click ‘Start Now’.

![image004](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/AmoCRM/image004.jpg)

3. When you have entered in your personal cabinet, please click ‘Settings’.

![image006](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/AmoCRM/image006.jpg)

4. Your API credentials: subdomain – for additional data.

![image008](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/AmoCRM/image008.jpg)

5. Your API credentials: e-mail  – for client ID.

![image010](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/AmoCRM/image010.jpg)

6. Your API credentials: API Key – for Client Secret.

![image012](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/AmoCRM/image012.jpg)
