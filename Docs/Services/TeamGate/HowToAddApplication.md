¹ Teamgate (Trial)  
Registration and Creating App  
Date: 26.09.2018.

1. Go to: (https://teamgate.com/)  
and click on the link "Start Free Trial".

![image002](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image002.png)

2. Fill all required fields and click on the button "Sign Up".

![image004](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image004.png)

3. Wait while the account is setting up.

![image006](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image006.png)

4. After setting up is done, you will be redirected to the CRM account page.

![image008](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image008.png)

5. Go to the profile menu.

![image010](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image010.png)

6. Select "Settings".

![image012](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image012.png)

7. In the next page select "Additional features"

![image014](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image014.png)

and then select "External Apps".

![image016](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image016.png)

Then click on the link "New App Key Request".

![image018](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image018.png)

8. Set App name and click on the button "CREATE".

![image020](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image020.png)

9. Select App.

![image022](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image022.png)

10. Copy your app_key to your App.

![image024](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image024.png)

# User's Auth Token

User must make a few steps for getting user’s auth token:

1. Select "My Profile" in the profile menu.

![image026](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image026.png)

2. Select "Integrations" in the left menu.

![image028](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image028.png)

3. Click "API Access" on the Tab menu.

![image030](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image030.png)

4. Copy the Auth Token.

![image032](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/Teamgate/image032.png)

Good Luck.
