# Google Drive registration

To register your application in Google Drive, follow the instructions below.

1. Go to the Google Developers Console by link:  (https://console.developers.google.com/),  
select the item: Google Cloud Storage. 

![image002](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image002.jpg)

The confirmation:

![image004](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image004.jpg)

2. Click on the button «Create a project».

![image006](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image006.jpg)

3. Confirm the creation of the project.

![image008](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image008.jpg)

4. Enter the name of your project (for example, we used the name bNesisTest) and click on the Create button. When you enter a project name, the Project ID is assigned automatically.

![image010](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image010.jpg)

5. Press the "Enable" button.

![image012](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image012.jpg)

6. Next we create credentials.

![image014](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image014.jpg)

7. Adding credentials.

![image016](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image016.jpg)

8. Creating an API key.

![image018](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image018.jpg)

9.

![image020](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image020.jpg)

10. Select OAuth client ID to create an Identifier and Secret client for OAuth authorization. Click on the button "Configure the access request window".

![image022](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image022.jpg)
![image024](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image024.jpg)

11. Obtaining the Customer ID and the customer secret. Select Application Type, Name, Allowed JavaScript Sources, Redirect URI. And click on the Create button.

![image026](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image026.jpg)

12. After the creation, a window with your client ID and Secret for OAuth authorization will be displayed.

![image028](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image028.jpg)

13. The client ID for OAuth 2.0 can be viewed by clicking on the name of this identifier.

![image030](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image030.jpg)

14. Copy your Customer ID and Client Secret to the bNesis application.

![image032](https://raw.githubusercontent.com/bNesisDeveloper/bNesis/master/Docs/Services/GoogleDrive/image032.jpg)
